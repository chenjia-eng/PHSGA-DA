1. How to install SPT-IFL

1. Run SPT-IFL from docker (Locally、Simply)

1.1.1. Make sure the machine contains a Docker environment.
1.1.2. Download the SPT-IFL and Add the data file to the project directory
git https://github.com/wertabc/SPT-IFL cd SPT-IFL # upload the data file (example: data.fasta)
1.1.3. Pull and build the environment image. (Time required)
sudo docker build -t pinc_images .
1.1.4. Create and Enter a new container.
sudo docker run -it pinc_images bash
1.1.5. Execute SPT-IFL for prediction.
python pinc.py -f data.fasta


1.2. Run SPT-IFL from source code (Complex)

1.2.1. Installation Environment(Autogluon、kentUtils).
1.2.2. Clone project, install related dependencies.
git clone https://github.com/wertabc/SPT-IFL cd SPT-IFL pip3 install -r requirements.txt
1.2.3. Execute SPT-IFL for prediction.
python pinc.py -f data.fasta
