1.Introduction

PHSGA‑DA is an intelligent platform for identifying plant heat‑stress‑responsive genes. By integrating multi‑level amino acid encoding features (CKSAAP, PseAAC, CKSAAGP) with Mamba‑gradient‑weighted domain adaptation technology, it can efficiently distinguish whether a plant protein is a heat‑stress‑responsive protein, and supports cross‑species recognition between model plants and non‑model crops.

2.PHSGA-DA online input

PHIGA‑DA accepts protein sequences in FASTA format.


2.1 Submission requirements
Format:Standard FASTA format. Sequence type:Protein amino acid sequences (20 standard letters: A, C, D, E, F, G, H, I, K, L, M, N, P, Q, R, S, T, V, W, Y). Size requirement: Less than 10000 lines in input box and no line limitation in batch model. Maximum allowable upload file size is 50 Mb. Name requirement: Sequence names beginning with ‘>’ symbol are required.


2.2 How to submit Protein sequences
There are two ways to submit protein sequences:
1) Single/few sequences: Paste FASTA‑formatted protein sequences into the large input box on the predict page, select “Limited Species” or “General Prediction”, and click Submit.
2) Batch sequences: Go to the Predict page, upload a .fasta file, and click Submit.


3. PHSGA-DA result output

For each input sequence, the system returns: ID: The sequence name (the part after > in FASTA). Length: The sequence length. Label: Positive = heat stress‑responsive gene, Negative = non-heat stress‑responsive gene score: Prediction confidence score (ranging from 0 to 1; the closer to 1, the more reliable the prediction result).

4. FAQ

4.1 How to retrieve results
When the user input sequence is provided to the network server for calculation, the network server will assign a unique task ID to this request. After the calculation is completed, users can enter the batch page and retrieve their results from the CPC server using the task ID


4.2 PHSGA-DA What are the functions
We trained a model based on the Mamba architecture, using the extracted features as input. It can efficiently identify cross‑species heat‑stress‑responsive genes and demonstrates strong generalization performance.
